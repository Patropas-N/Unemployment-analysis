# Unemployment Analysis in India (2019–2020)

## Project Overview

This project performs an Exploratory Data Analysis (EDA) on unemployment trends across India using Python.

## Objectives

- Analyze unemployment trends across different states.
- Compare unemployment between 2019 and 2020.
- Compare rural and urban unemployment.
- Study monthly unemployment patterns.
- Explore relationships between unemployment, employment, and labour participation.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset

The project uses two unemployment datasets containing:
- State/Region
- Date
- Estimated Unemployment Rate
- Estimated Employment
- Labour Participation Rate
- Rural/Urban Area

## Analysis Performed

- Data Cleaning
- Missing Value Handling
- Feature Engineering
- Regional Analysis
- Monthly Trend Analysis
- Yearly Trend Analysis
- Correlation Analysis
- Heatmaps
- Data Visualization

## Key Findings

- Unemployment increased substantially in 2020.
- Urban unemployment was higher than rural unemployment.
- Tripura and Haryana recorded the highest unemployment rates.
- COVID-19 had a significant impact on unemployment trends.
